Объектілерді құру, мұрагерлік

Негізгі класс (Animal)
Ұрпақ класы (Crocodile)

Түпнұсқасы: inheritance.PNG

Объектіні құру алгоритмі:

1) 1-ші Негізгі класстың статикалық айнымалылары инициализацияланады
2) 2-ші Ұрпақ класының статикалық айнымалылары инициализацияланады
3) 3-ші Негізгі класстың статикалық емес айнымалылары инициализацияланады
4) 4-ші Негізгі класстың конструкторы шақырылады
5) 5-ші Ұрпақ класының статикалық емес айнымалылары инициализацияланады
6) 6-ші Ұрпақ класының конструкторы шақырылады

Мысал: Animal class, Crocodile class
